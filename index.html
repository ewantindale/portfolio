Workout Tracker Mobile App

  • View Source Code

Technologies: React Native, Redux, Firebase, NoSQL, CSS

  • Guides the user through a workout designed to build strength and muscle, as well as logging their workouts to track their progress over time.
  • Serverless app, using React Native/Redux on the frontend and communicating directly with Firebase for authentication and data storage.
  • Uses Google Signin with Firebase Authentication for a quick and easy sign up process.
  • User data is stored using Cloud Firestore, an offline-first NoSQL database allowing the app to be used with or without an internet connection.
